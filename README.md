# Comics Finder App

> My personal web app to manage my comics library.

This app is built using SvelteKit, DaisyUI and Prisma.

## Environment variables

| Name                  | Value        |
| --------------------- | ------------ |
| HOST                  | 0.0.0.0      |
| PORT                  | 3000         |
| ORIGIN                | _Vercel URL_ |
| SESSION_COOKIE_SECURE | Off          |

## Deployment

This app is deployed using Vercel.
